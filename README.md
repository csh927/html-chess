# html-chess


# ♔ Grandmaster Chess - Premium Experience

이 프로젝트는 최신 웹 기술과 현대적인 디자인 감각을 결합하여 제작된 **고급형 체스 게임**입니다. 리체스(Lichess)의 전문적인 아키텍처 패턴을 참고하여 견고한 로직과 유려한 UI를 갖추고 있습니다.

---

## ✨ 주요 기능 (Key Features)

### 1. 전문적인 체스 엔진 (Professional Logic)
*   **OOP 기반 설계**: `ChessEngine`(로직)과 `ChessUI`(화면) 클래스로 분리된 유지보수성이 높은 구조입니다.
*   **완벽한 체스 규칙**: 일반적인 이동 뿐만 아니라 캐슬링(Castling), 앙파상(En Passant), 폰 프로모션(Promotion) 등의 특수 규칙을 완벽하게 지원합니다.
*   **실시간 판정**: 체크(Check) 및 체크메이트(Checkmate) 상황을 실시간으로 감지합니다.

### 2. 프리미엄 사용자 인터페이스 (Premium UI)
*   **글래스모피즘(Glassmorphism)**: 투명도와 블러 효과를 활용한 세련된 레이아웃을 제공합니다.
*   **커스텀 타이머**: 불렛(1분), 블리츠(3분/10분) 등 프리셋 제공 및 **사용자 지정 시간(분)** 설정 기능이 포함되어 있습니다.
*   **대국 기록(Move History)**: 전문 체스 데이터베이스 스타일의 기보 창을 통해 대국 흐름을 한눈에 파악할 수 있습니다.
*   **시각적 보조**: 마지막 수(Last Move) 하이라이트 기능과 이동 가능한 칸 가이드 애니메이션을 제공합니다.

### 3. 부가 기능
*   **무르기(Undo)**: 대국 중 실수를 만회할 수 있는 무르기 기능을 지원합니다.
*   **사운드 효과**: 이동, 기물 잡기, 체크 시 각각 다른 사운드 피드백을 제공합니다.

## 🛠 기술 스택 (Tech Stack)

*   **HTML5**: 시맨틱 웹 구조 및 게임 레이아웃
*   **CSS3**: 글래스모피즘 변수, 그리드 시스템, 미디어 쿼리(반응형)
*   **JavaScript (ES6+)**: 클래스 기반 엔진 설계, 웹 오디오 API, 드래그 앤 드롭 API
*   **Icons**: Google Material Symbols Rounded

## 🚀 실행 방법 (How to Run)

본 프로젝트는 의존성 없는 **순수 웹 기술(Vanilla Stack)**로 제작되었습니다. 별도의 설치 과정 없이 다음 단계를 통해 바로 실행할 수 있습니다.

1.  본 폴더의 `index.html` 파일을 더블 클릭하거나 브라우저(Chrome, Edge 등)로 드래그합니다.
2.  브라우저를 통해 프리미엄 체스 게임을 즉시 즐기실 수 있습니다.

## 📁 파일 구조 (File Structure)

*   `index.html`: 게임의 구조, 네비게이션, 모달 및 UI 레이아웃 정의
*   `style.css`: 테마, 색상 변수, 레이아웃 및 애니메이션 스타일 정의
*   `script.js`: 체스 엔진 알고리즘 및 UI 상호작용 로직
*   `README.md`: 프로젝트 설명 및 매뉴얼
